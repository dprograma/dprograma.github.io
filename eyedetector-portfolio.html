Overview

This project is a computer-vision system that detects a person’s eye state — open or closed — and infers drowsiness in real time from a live camera feed. Its target use case is driver-safety: raising an alert when signs of fatigue are detected.

It combines face and eye detection with a trained classifier, processing frames in real time to trigger timely warnings.

Key Features

  • Real-time eye-state classification
  • Driver drowsiness detection
  • Face & eye region detection
  • Live camera-feed processing
  • Safety alerting on fatigue
  • Deep-learning classifier
