Overview

Eunixma is an online store and marketplace offering fashion, gadgets and everyday essentials. I built it as a Laravel backend with a React frontend — a decoupled architecture exposing a clean REST API to the storefront.

The platform covers the full commerce journey: product catalogue and search, cart and checkout, order management and an admin dashboard for inventory and fulfilment.

Key Features

  • Product catalogue with search & categories
  • Cart, checkout & order management
  • Admin dashboard for inventory & orders
  • Laravel REST API consumed by React SPA
  • Secure authentication & role-based access
  • Responsive storefront UI
