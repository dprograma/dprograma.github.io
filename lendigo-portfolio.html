Overview

Lendigo is a financial-technology firm on a mission to become a leading SME-focused institution in Nigeria through technology and innovation. I worked as a Fullstack Python Developer on the digital lending platform that has disbursed over ₦6 billion in short-term business loans to 2,500+ businesses.

I built Django backend services for loan origination and lifecycle management — automated credit-assessment workflows, loan state machines and repayment scheduling — plus the RESTful APIs powering the web and mobile application flow, reducing application-to-decision time by 40%.

Key Features

  • Loan origination & lifecycle state machines
  • Automated credit-assessment workflows
  • KYC/AML pipeline (BVN, identity services)
  • Real-time credit-risk scoring engine
  • Automated disbursement & repayment reconciliation
  • React.js borrower dashboard & document portal
  • RESTful APIs for web & mobile flows
  • CI/CD with GitHub Actions, PostgreSQL warehouse
