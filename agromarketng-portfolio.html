Overview

AgromarketNG is a classified-ads marketplace for the agricultural sector — connecting farmers, buyers and suppliers to trade produce, inputs and equipment. It brings the classifieds model to Nigerian agriculture with category browsing, listings and search.

Built with a Django backend and React client, it supports listing management, media uploads and location-aware discovery.

Key Features

  • Agricultural listings & categories
  • Search & location-aware discovery
  • Listing management & media uploads
  • Buyer–seller connection
  • Django REST backend
  • React client-side experience
